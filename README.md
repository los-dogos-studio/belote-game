გურული ბელოტის სრული წესები

1. ზოგადი აღწერა
   გურული ბელოტი ითამაშება 4 მოთამაშით, რომლებიც იყოფიან 2 გუნდად.
   პარტნიორები პირისპირ სხედანგამოიყენება 32-კარტიანი დასტა: 7, 8, 9, 10, ვალეტი, დამა, მეფე, ტუზითამაშის მიზანია 1001 ქულის დაგროვებათამაში მიმდინარეობს რაუნდებად, სადაც ერთ-ერთი გუნდი ხდება მკოზირებელი, ხოლო შემდეგ ითვლება, შეასრულა თუ არა მან ვალდებულება.

2. მოთამაშეთა განლაგება და კარტის დარიგება
   მოთამაშეები სხედან წრიულად ისე, რომ პარტნიორები პირისპირ აღმოჩნდნენ. კარტის დარიგება ხდება საათის ისრის მიმართულებით — ყოველი რაუნდის შემდეგ დამრიგებელი იცვლება.
   2.1. პირველი დარიგება
   დამრიგებელი თითოეულ მოთამაშეს ურიგებს 5-5 კარტს.
   ამის შემდეგ ერთი კარტი ამოტრიალდება, რომელიც წარმოადგენს პოტენციურ კოზირს.

3. კოზირის განსაზღვრა
   3.1. ავტომატური კოზირი — ვალეტის წესი
   თუ დამრიგებლის შემდეგ მოთამაშეს (მარცხნივ მჯდომს) ხელში აქვს ამოტრიალებული ფერის ვალეტი:
   ეს ფერი ავტომატურად ხდება კოზირიეს მოთამაშე ხდება მკოზირებელიკოზირის არჩევის პროცესი სრულდება3.2. პირველი კრუგი
   თუ ავტომატური კოზირი არ დადგა, დამრიგებლის შემდეგ მოთამაშიდან დაწყებული ყველა რიგრიგობით ამბობს — სურს თუ არა ამოტრიალებული ფერის კოზირად აღება.
   თუ რომელიმე მოთამაშე თანხმდება:
   ის ხდება მკოზირებელიამ კარტის ფერი ხდება კოზირი3.3. მეორე კრუგი
   თუ პირველ კრუგში ყველამ უარი თქვა, იწყება მეორე კრუგი, სადაც მოთამაშეებს შეუძლიათ გამოაცხადონ ნებისმიერი სხვა ფერი კოზირად, გარდა თავდაპირველად ამოტრიალებული ფერისა.
   3.4. დამრიგებლის ვალდებულება
   თუ მეორე კრუგშიც ყველამ უარი თქვა, დამრიგებელს უარის თქმა აღარ შეუძლია — ვალდებულია აირჩიოს ფერი და გახდეს მკოზირებელი.

4. დარჩენილი კარტების დარიგება
   კოზირის განსაზღვრის შემდეგ ამოტრიალებული კარტი ყოველთვის მიეცემა მკოზირებელს — იმის მიუხედავად, პირველ კრუგში დადგა კოზირი თუ მეორეში.
   შემდეგ ნაწილდება დარჩენილი კარტები:
   ჩვეულებრივ მოთამაშეებს ურიგდებათ 3-3 კარტიმკოზირებელს ურიგდება 2 კარტი (ამოტრიალებული უკვე აქვს)საბოლოოდ ყველას აქვს 8 კარტი.

5. კომბინაციების გამოცხადება
   კომბინაციები უნდა გამოცხადდეს და აჩვენონ მე-2 ხელზე (მეორე ტრიკი). თუ მოთამაშეს დაავიწყდა ჩვენება, მოწინააღმდეგეს შეუძლია გააპროტესტოს და კომბინაცია აღარ ჩაითვლება.

6. კომბინაციები და მათი ქულები
   6.1. ბელოტი
   თუ მოთამაშეს აქვს კოზირის მეფე + კოზირის დამა — ეს არის ბელოტი და იძლევა 20 ქულას.
   ქულა ეთვლება მხოლოდ სწორი გამოცხადებისას:
   პირველ კარტზე: „ბელოტი"მეორეზე: „რებელოტი"გამოცხადება შეიძლება ნებისმიერ ხელზე, როცა შესაბამის კარტს თამაშობს. თუ დაავიწყდა — არ ითვლება.
   6.2. ზედიზედ კარტები (მიმდევრობა)

7. რომელი გუნდის კომბინაციები ითვლება
   ორივე გუნდის კომბინაციები ერთდროულად არ ითვლება — ითვლება მხოლოდ უფრო ძლიერი კომბინაციის მქონე გუნდისა.
   შედარების წესი (ერთნაირი სიგრძის მიმდევრობებისთვის)
   კოზირის ფერში შესრულებული კომბინაცია იმარჯვებსთუ ორივე/არცერთი კოზირი არ არის → უფრო მაღალი კარტით შემდგარი იმარჯვებსთუ ეს გათანაბრებაც — საათის ისრის მიმართულებით პირველი მოთამაშის კომბინაცია იმარჯვებს„ოთხი ერთნაირი" VS „5-კარტიანი მიმდევრობა"
   თუ 5-კარტიანი მიმდევრობა კოზირშია → ის იმარჯვებსსხვა შემთხვევაში → საათის ისრის მიმართულებით პირველი მოთამაშის კომბინაცია იმარჯვებს

10. სვლის წესები
    10.1. პირველი სვლა
    დამრიგებლის შემდეგ მოთამაშე იწყებს პირველ ხელს.
    10.2. ცვეტის აყოლა სავალდებულოა
    თუ მოთამაშეს აქვს გახსნილი ფერის კარტი — აუცილებლად უნდა ჩამოვიდეს იმავე ფერში. „მოტყუება" დაუშვებელია.
    10.3. გაჭრა სავალდებულოა
    თუ მოთამაშეს გახსნილი ფერი არ აქვს — აუცილებლად უნდა გაჭრას კოზირით, თუ შეუძლია. არ აქვს მნიშვნელობა პარტნიორი იგებს თუ არა — ყველა ვალდებულია გაჭრას.
    10.4. მაღალი კოზირის დადება
    თუ ხელი უკვე კოზირით არის გაჭრილი:
    თუ მოთამაშეს უფრო მაღალი კოზირი ჰყავს → აუცილებლად უნდა დაარტყასთუ მაღალი კოზირი არ ჰყავს → შეუძლია დადოს ნებისმიერი კოზირი
11. ხელის მოგება
    ხელს იგებს:
    გახსნილი ფერის ყველაზე ძლიერი კარტი (თუ ხელი არ გაჭრილა)ყველაზე ძლიერი კოზირი (თუ ხელი გაჭრილია)შემდეგ ხელს იწყებს წინა ხელის გამარჯვებული.

12. ქულების დათვლა
    ერთ რაუნდში სულ არის 162 ქულა (მათ შორის ბოლო ხელის 10 ქულა).
    ჯერ ითვლება: კარტების ქულები + კომბინაციები, შემდეგ ხდება დამრგვალება.
    დამრგვალება
    დასრულება მიმართულება     5-ზე ნაკლები ქვემოთ   ზუსტად 5 ქვემოთ   5-ზე მეტი ზემოთ   მაგალითები: 84 → 80 / 85 → 80 / 86 → 90

13. ჩავარდნა
    თუ მკოზირებელი გუნდი ვერ აგროვებს მოწინააღმდეგეზე მეტ ქულას (ტოლი არ ჩაითვლება ჩავარდნად):
    მკოზირებელ გუნდს ეწერება −50მოწინააღმდეგეს ეწერება 160მკოზირებელი გუნდის კომბინაციებიც მოწინააღმდეგეს გადაეწერება
14. კაპო
    თუ რომელიმე გუნდი ყველა ხელს წაიყვანს, ეს არის კაპო.
    კაპოს გუნდი იღებს 250 ქულასმეორე გუნდს ეთვლება მხოლოდ საკუთარი კომბინაციების ქულები (თუ ჰქონდა)
15. თანაბარი ქულა (81 – 81)
    თუ ორივე გუნდი დააგროვებს ზუსტად 81 ქულას:
    არცერთ გუნდს არ ეწერება ქულაქულები „იყინება" და გადადის შემდეგ რაუნდზეშემდეგ რაუნდში გამარჯვებული გუნდი ამ გაყინულ ქულებსაც მიიღებსმაგალითი:
    რ. 1: A = 81 / B = 81 → ტოლი → 81 ქულა „ჰაერში" რ. 2: A = 90 / B = 72 → A იგებს შედეგი: A-ს ეწერება 90 + 81 = 171 / B-ს ეწერება 72
16. თამაშის დასრულება
    თამაში გრძელდება მანამ, სანამ ერთ-ერთი გუნდი არ დააგროვებს 1001 ქულას.
